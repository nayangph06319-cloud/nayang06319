คู่มือ V10 Admin Full Control

สิ่งที่เพิ่ม:
1. หน้า admin.html เพิ่ม/แก้ไข/ลบ/ซ่อน/เผยแพร่ ได้
2. อัปโหลดรูปและ PDF ไป Google Drive ได้
3. รูปภาพใช้ลิงก์ Google Drive แบบแสดงบนเว็บได้
4. หน้าเว็บประชาชนจะไม่แสดงรายการที่ตั้งสถานะ hidden

วิธีติดตั้ง:
1. เปิด Google Apps Script
2. ลบ Code.gs เดิม แล้ววาง Code.gs จากชุด V10
3. แก้ค่า 3 บรรทัดบนสุด:
   - SPREADSHEET_ID
   - DRIVE_FOLDER_ID
   - ADMIN_PASSWORD
4. Deploy > Manage deployments > รูปดินสอ > Version: New version > Deploy
5. ตรวจ cms-config.js ให้ WEB_APP_URL เป็นลิงก์ /exec ล่าสุด
6. อัปโหลดไฟล์ทั้งหมดของ V10 ขึ้น GitHub Pages/Netlify
7. เปิด /admin.html เพื่อจัดการข้อมูล

การใช้งาน:
- เพิ่มข่าว: เข้า admin.html > ข่าวสาร > กรอกข้อมูล > บันทึก/เผยแพร่
- แก้ไข: กดปุ่ม แก้ไข ที่รายการเดิม > แก้ข้อมูล > บันทึก
- ซ่อน: กดปุ่ม ซ่อน รายการจะหายจากหน้าเว็บ แต่ยังอยู่ใน Sheet
- เผยแพร่: กดปุ่ม เผยแพร่ รายการจะกลับมาแสดง
- ลบ: กดปุ่ม ลบ รายการจะถูกลบจาก Google Sheet
- อัปโหลดรูป/PDF: ลากไฟล์วางในกล่องอัปโหลด ระบบจะนำลิงก์ไปใส่ช่องให้

หมายเหตุ:
ถ้ารายการเก่าที่เคยเพิ่มก่อน V10 ไม่มี id ปุ่มแก้/ลบอาจไม่ทำงาน แนะนำเพิ่มใหม่ผ่าน V10 หรือเติมคอลัมน์ id ใน Sheet เอง

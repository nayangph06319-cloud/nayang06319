V12 Enterprise Auto Setup - รพ.สต.นายาง

หลังติดตั้งครั้งแรก ไม่ต้องแก้โค้ดด้วย VS Code อีก

วิธีใช้:
1. อัปโหลดไฟล์ V12 ทั้งหมดขึ้น GitHub Pages หรือ Netlify
2. เปิด Google Apps Script แล้ววาง Code.gs จากชุดนี้
3. Deploy > Web app
   Execute as: Me
   Who has access: Anyone
4. คัดลอก Web App URL ที่ลงท้าย /exec
5. เปิด https://ชื่อเว็บ/setup.html
6. วาง Web App URL, Google Sheet URL, Google Drive Folder URL และตั้งรหัสผ่าน Admin
7. กดติดตั้งระบบ
8. เปิด https://ชื่อเว็บ/admin.html เพื่อใช้งาน

หลังจากนั้น:
- เพิ่ม/แก้ไข/ลบ/ซ่อน/เผยแพร่ข่าว
- อัปโหลดรูปภาพ/PDF
- จัดการประกาศ กิจกรรม บุคลากร บริการ ตารางบริการ
ผ่านหลังบ้านได้ทั้งหมด

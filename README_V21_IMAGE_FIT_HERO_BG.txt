V21 Image Fit + Hero Background

ปรับปรุง:
1. เปลี่ยนภาพพื้นหลังหน้าแรกเป็นภาพอาคาร รพ.สต.นายางที่อัปโหลด
2. เพิ่ม overlay อ่อน ๆ ให้ตัวอักษรอ่านง่าย
3. แก้ภาพข่าว/กิจกรรม/Banner/Gallery ไม่ให้ล้นกรอบ
4. ใช้ object-fit: cover และกำหนดความสูงเหมาะสม
5. รองรับมือถือ ภาพไม่ยืด ไม่แตก ไม่ล้นพื้นที่

วิธีอัปเดต:
1. อัปโหลดไฟล์ V21 ทั้งหมดทับของเดิม
2. ถ้าใช้ GitHub/Netlify ให้ลากไฟล์ทั้งหมดขึ้นใหม่
3. เปิดหน้าเว็บแล้วกด Ctrl+F5
หมายเหตุ: ไม่จำเป็นต้อง Deploy Apps Script ใหม่ ถ้าไม่ได้เปลี่ยน Code.gs

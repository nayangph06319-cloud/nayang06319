V15 Enterprise Government Edition

เพิ่มจาก V14:
- Banner Slide พร้อมวันเริ่ม/สิ้นสุด
- Popup ประชาสัมพันธ์
- Dashboard KPI ผู้บริหาร
- นัดหมายออนไลน์ / ร้องเรียนออนไลน์ / E-Service
- Multi-user schema
- Audit Log / Backup Log
- ปุ่ม Backup Now สำรอง Google Sheet ไปยัง Drive
- รองรับเชื่อม Home BP, NCD by Nayang, Smart VHV ผ่าน AppScript Hub

วิธีอัปเดต:
1. วาง Code.gs จาก V15 ใน Apps Script
2. Deploy > New Version
3. อัปโหลดไฟล์ V15 ทั้งหมดขึ้น GitHub/Netlify
4. เปิด admin.html จัดการข้อมูล

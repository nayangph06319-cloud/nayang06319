V20 Government Professional UI

ปรับหน้าเว็บใหม่ให้เป็นมืออาชีพ:
- Header สวยขึ้น พร้อม Top strip
- เมนูจัดหมวดหมู่ Drop-down
- Hero section ใหม่
- Quick Service / Banner / KPI / ข้อมูลหน่วยงาน / ข่าว / บริการ / อสม. / AppScript / E-Service / Video / Download
- Responsive มือถือ
- ข้อมูลยังดึงจากหลังบ้านทั้งหมด
- ลดปัญหา cache ด้วย _ts ตอนโหลด Apps Script

วิธีอัปเดต:
1. วาง index.html/admin.html/setup.html ทับไฟล์เดิม
2. วาง Code.gs จากชุดนี้ใน Apps Script แล้ว Deploy New Version
3. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub/Netlify
4. เปิดหน้าเว็บแล้วกด Ctrl+F5
